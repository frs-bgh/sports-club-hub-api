Sports Club Hub API

Node.js 20 + Express + SQLite (Prisma)

Base URL

/api

Health

  • GET /api/health

Members (CRUD)

  • GET /api/members?limit=10&offset=0
  • GET /api/members?search=fariss
  • GET /api/members/:id
  • POST /api/members
  • PUT /api/members/:id
  • DELETE /api/members/:id

News (CRUD)

  • GET /api/news?limit=10&offset=0
  • GET /api/news?search=club
  • GET /api/news/:id
  • POST /api/news
  • PUT /api/news/:id
  • DELETE /api/news/:id

Example

POST /api/members
{
  "firstName": "John",
  "lastName": "Doe",
  "age": 22,
